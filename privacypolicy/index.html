1. 基本方針

Record Recorder（以下「本アプリ」）は、アナログレコードのコレクションを管理するためのiPhoneアプリです。

本アプリは、ユーザーのプライバシーを最大限尊重します。収集するデータは最小限にとどめ、すべてのデータはお使いのデバイス内のみに保存されます。

本アプリは外部サーバーへのデータ送信を行いません。ユーザーデータが外部に漏洩する仕組みはありません。

2. 収集・使用する情報

本アプリが取り扱う情報は以下のとおりです。

情報の種類 利用目的 保存場所 外部送信
カメラで撮影した画像 ジャケット画像の登録・OCR文字認識 デバイス内のみ なし
フォトライブラリの画像 ジャケット画像の選択・登録 デバイス内のみ なし
バーコード情報（EAN-13） Discogs APIでのレコード情報検索 一時的な処理のみ Discogs APIへ送信
レコード情報（タイトル等） Discogs APIでのレコード情報検索 デバイス内のみ 検索時にDiscogs APIへ送信
コレクションデータ 登録済みレコードの管理・表示 デバイス内のみ なし
Discogs API への送信は、バーコードまたはテキストによるレコード情報の検索時のみ発生します。ユーザーの個人情報は含まれません。

3. 収集しない情報

本アプリは以下の情報を一切収集しません。

  • 氏名・住所・メールアドレスなどの個人識別情報
  • 位置情報（GPS）
  • 端末の識別情報（デバイスID・広告ID）
  • 利用状況の分析・トラッキングデータ
  • クラッシュレポートや診断データ

4. アクセスする端末の権限

本アプリは以下の権限を使用します。権限の利用はユーザーの許可を得た場合のみです。

権限 利用目的
カメラ ジャケット画像の撮影およびバーコードのスキャン
フォトライブラリ（読み取り） 既存の写真からジャケット画像を選択する
フォトライブラリ（書き込み） 将来の機能拡張に備えた権限（現バージョンでは使用しない場合があります）

これらの権限は、各機能を利用する際にiOSの標準ダイアログで許可を求めます。許可しない場合でも、該当機能を除いたアプリの基本機能は利用できます。

5. 第三者サービスについて

本アプリは以下の第三者サービスを利用します。

Discogs API

バーコードまたはテキストからレコード情報を検索するために使用します。検索時に検索キーワード（バーコード番号またはテキスト）がDiscogs社のサーバーに送信されます。ユーザーの個人情報は含まれません。

  • 運営: Discogs LLC（米国）
  • プライバシーポリシー: https://support.discogs.com

Apple Music（URLスキーム）

登録済みレコードをApple Musicで検索する際に、music://スキームを使用してApple Musicアプリを起動します。本アプリからAppleへの情報送信は発生しません。

6. データの保存と削除

本アプリで登録したすべてのデータ（レコード情報・ジャケット画像）は、お使いのiPhoneのローカルストレージ（SwiftData）に保存されます。

  • データは外部サーバーには送信・保存されません
  • アプリを削除することで、すべてのデータが完全に削除されます
  • iOSのバックアップ機能を使用した場合、バックアップ先にデータが含まれます

7. 広告について

本アプリは広告を表示しません。広告SDKの組み込みはありません。

8. 子どものプライバシー

本アプリは13歳未満の子どもを対象としていません。13歳未満の方から意図的に個人情報を収集することはありません。

9. プライバシーポリシーの変更

本ポリシーを変更する場合は、このページを更新し、ページ上部の「最終更新」日付を改定します。重要な変更がある場合は、アプリのアップデートにてお知らせします。

Privacy Policy (English)

Effective Date: May 1, 2026

Record Recorder is an iPhone app for managing analog record collections. We are committed to protecting your privacy.

Data We Collect

  • Camera images — used locally for jacket photos and OCR. Not transmitted externally.
  • Photo library images — used locally for jacket photos. Not transmitted externally.
  • Barcode / search text — sent to the Discogs API only during record searches. Contains no personal information.
  • Collection data — stored locally on your device only. Never uploaded to any server.

Data We Do NOT Collect

  • Personal identifiers (name, email, address)
  • Location data
  • Device identifiers or advertising IDs
  • Analytics or tracking data

Third-Party Services

We use the Discogs API to search for record information. Only search keywords (barcode or text) are sent. No personal data is included. See Discogs Privacy Policy.

For privacy-related inquiries, please contact: contact@dragonsystem.net
